# 贺州营岳建材有限公司 - 企业官网

## 项目简介

本项目为贺州营岳建材有限公司的官方网站，包含企业介绍、产品展示、联系方式等核心功能模块。网站采用响应式设计，适配PC、平板和手机等多种终端设备。

## 技术架构

### 前端技术
- HTML5
- CSS3
- JavaScript
- Font Awesome 图标库

### 设计特点
- 响应式布局
- 现代化UI设计
- 交互动效
- 无障碍访问
